How to Fix Wi-Fi Not Connecting on Windows or Phones

Posted on October 01, 2025 • by Ibrahim

Few things are more annoying than your Wi-Fi refusing to connect when you need it most. Luckily, most problems come down to simple settings or quick resets. Here’s a guide to get you back online, whether you’re on Windows or a phone.

1. Check the Basics First

  • Airplane mode: Make sure it’s off.
  • Wi-Fi switch: On laptops, confirm the Wi-Fi button or shortcut (like Fn + F2) isn’t toggled off.
  • Router lights: If the Wi-Fi light on your router isn’t blinking, the issue may be with your internet provider, not your device.

2. Restart Everything

Restart your computer or phone. Then restart your router by unplugging it for 30 seconds and plugging it back in. This alone fixes many “can’t connect” problems.

3. Forget and Reconnect to the Network

Sometimes your saved Wi-Fi profile gets corrupted.

On Windows:

  1. Click the Wi-Fi icon on the taskbar.
  2. Right-click your network → Forget.
  3. Reconnect by entering the password again.

On Android/iPhone:

  1. Go to Settings → Wi-Fi.
  2. Tap your network → Forget/Remove.
  3. Reconnect and re-enter the password.

4. Run Troubleshooters (Windows)

Press Windows Key + I → System → Troubleshoot → Other troubleshooters. Run the Internet Connections troubleshooter. Windows will attempt to diagnose and fix the problem automatically.

5. Check IP and DNS Settings

On Windows:

  1. Press Windows Key + R, type ncpa.cpl, hit Enter.
  2. Right-click your Wi-Fi adapter → Properties.
  3. Double-click Internet Protocol Version 4 (TCP/IPv4).
  4. Make sure “Obtain an IP address automatically” and “Obtain DNS server automatically” are selected.

On Phones: Go to Wi-Fi → your network → Advanced/Modify settings, and reset IP/DNS to automatic if they’re custom.

6. Update Network Drivers (Windows)

  1. Press Windows Key + X → Device Manager.
  2. Expand Network adapters.
  3. Right-click your Wi-Fi card → Update driver → Search automatically.

7. Reset Network Settings

On Windows:

  1. Press Windows Key + I → Network & Internet → Advanced network settings.
  2. Scroll down to Network reset → Reset now.

On Android/iPhone:

  • Android: Go to Settings → System → Reset options → Reset Wi-Fi, mobile & Bluetooth.
  • iPhone: Go to Settings → General → Transfer or Reset iPhone → Reset → Reset Network Settings.

8. Check for Hardware Issues

Try connecting another device to the same Wi-Fi. If no device works, it’s your router or ISP. If other devices connect fine, your computer or phone’s Wi-Fi adapter may be faulty.

Final Thoughts

Wi-Fi issues can be frustrating, but in most cases they’re solved by restarting, forgetting the network, or resetting settings. If problems persist, it may be time to contact your ISP or check for hardware failure.
